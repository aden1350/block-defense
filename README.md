# 🤖 Block Defense - 方块塔防游戏

一个类似俄罗斯方块的塔防游戏，防御塔从天上掉落，怪物进攻中间平台。

## 🎮 游戏玩法

1. **控制方块**：使用键盘左右移动和旋转
2. **堆叠防御塔**：不同颜色方块=不同类型防御塔
3. **击退怪物**：怪物从两侧进攻平台
4. **存活**：保护平台生命值

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 左右移动方块 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 放置方块 |

## 🏗️ 防御塔类型

| 颜色 | 名称 | 技能 |
|------|------|------|
| 🟡 黄色 | 炮塔 | 高伤害 |
| 🔵 蓝色 | 冰塔 | 减速 |
| 🟣 紫色 | 电塔 | 链式攻击 |
| 🟠 橙色 | 箭塔 | 快速攻击 |
| 🔴 红色 | 核塔 | 范围伤害 |

## 🚀 开始游戏

直接在浏览器中打开 `game_prototype.html` 即可开始游戏！

```bash
# 或者用Python启动本地服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```
